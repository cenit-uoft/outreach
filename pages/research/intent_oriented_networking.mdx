## ION: Intent-Oriented Networking 

# ![img](/research/ion.png)

We study networking technologies to allow end users and processes to state the intent of what service the Internet or a network infrastructure should provide for them with a high-level description, enable the network to automatically map user requirements to performance and policy criteria, and have the network deliver the best performance that satisfies user requirements taking into account network conditions.
